# Codabench Competition: Credit Scoring

- Ссылка на соревнование: https://www.codabench.org/competitions/9892/
- Финальный результат: **Топ-6**
- Метод решения: Catboost + feature engineering и SHAP анализ
- Дата: 08.2025


## О проекте
Задача заключается в бинарной классификации суждения о выплаченном (0) или невыплаченном займе (1), метрика оценки ROC-AUC


## Состав репозитория
- 'competition.ipynb' - основной файл с решением
- 'requirements.txt' - список зависимостей
- 'README.md' - описание

## Как запустить
1. Установить зависимости ('pip install -r requirements.txt')
2. Запустить ноутбук
3. Скачать датасеты с платформы и положить в корневую папку с основным файлом
4. Выполнить все ячейки до конца, в результате чего получится финальный файл submission.csv соответствующего формата для отправки

## Финальный результат:
- топ-6 из 192 участников
- Принятый roc-auc: 0.78
